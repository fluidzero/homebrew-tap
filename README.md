# FluidZero Homebrew Tap

Homebrew formulae for [FluidZero](https://fluidzero.ai) tools.

## Installation

```bash
brew tap fluidzero/tap
brew install fz
```

Or in a single command:

```bash
brew install fluidzero/tap/fz
```

## Formulae

| Formula | Description |
|---------|-------------|
| `fz` | CLI for the FluidZero document intelligence platform |
